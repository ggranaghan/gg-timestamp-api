Timestamp API

How to Use

Pick an option below. In the address bar, type a custom query string containing a date after the route path. You will be directed to a page with JSON. Or, click the button to recive the current timestamp.

Use for current timestamp:

Or enter a valid unix date and submit:

Example Input:

  • /api/time/2021-01-29
  • /api/time/1611957466207

Example Output:

{"unix":1451001600000,
"utc":"Fri, 25 Dec 2015 00:00:00 GMT"}
